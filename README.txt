INSTRUCCIONES PARA EJECUCIÓN:
Incluir las siguientes librerías al momento de ejecutar
- LWJGL:
https://www.lwjgl.org/customize
- Commons IO:
https://commons.apache.org/proper/commons-io/
- IDE Eclipse cualquier versión.

En eclipse:
1) Click derecho al proyecto
2) Build Path -> Add Libraries -> User Library
3) Crear la carpeta de la librería y añadir todos los .jar de cada una
